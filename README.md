# COVID 19 plots for Croatia

This repository contains the code used for generating plots tracking the progress of COVID-19 cases in Croatia displayed on [ppalasek.github.io](https://ppalasek.github.io/).

All plots are in Croatian, apologies about that.

Some of the plots are also saved as interactive plots using ggplotly and htmlwidgets. These are saved into the ./html/ folder.

**Feel free to use the code to generate your plots, but please don't forget to give credit in case you do. Thanks!**

## Usage

* Install all dependencies (TODO: list them).
* Download Croatian map data from https://www.diva-gis.org/gdata and save into data/HRV_adm/.
* Run ./do_all.sh
  * This checks whether new data has been published on koronavirus.hr (https://www.koronavirus.hr/json/?action=po_danima_zupanijama_zadnji', using check_covid_data.py)
  * get_data.r gets the data from koronavirus.hr, generate_region_data.r generates data per region. The data is stored at:
    * data/latest/last_data_po_danima_zupanijama.json
    * data/latest/last_data_po_osobama.json
    * data/latest/region_diff_cases_df.csv
    * data/latest/region_diff_deaths_df.csv
  * The plots are generated by different scripts. See below for more detail.
  * After all the plots are generated, the index.md file is generated using generate_html.r.
  * Tracking is added using add_tracking.py to all html files.
  * [publish]
* That's it.

### Data

The cases/deaths data is fetched from the json files on koronavirus.hr:

* https://www.koronavirus.hr/json/?action=po_danima_zupanijama_zadnji
* https://www.koronavirus.hr/json/?action=po_osobama

Some data in the json files was wrong so some parts are read from

* data/missing_data.json. I do not guarantee that the data in this file is 100% correct. The active cases data is wrong for sure, as here only the cases and death data are used.



The hospitalisation/number of tests/number of people on respirators is downloaded from https://www.hzjz.hr/aktualnosti/covid-19-izvjesce-hzjz-a/ and read from:

* data/latest/last_hzjz_data.csv.



The map data is downloaded from https://www.diva-gis.org/gdata and read from data/HRV_adm/.



The population data was downloaded from dzs.hr (2019 data). It is stored at:	

* data/cro_population_by_age.csv
* data/cro_population_by_age_and_sex.csv



The vaccination data is manually entered from https://www.hzjz.hr/aktualnosti/covid-19-izvjesce-hzjz-a/ and read from:

* data/vaccination/

## Trends per region (with exponential fit) 

* Generated using fit_exp_all.py.

(Trend kretanja 14-dnevnog broja slučajeva na 100k stanovnika opisan eksponencijalnom krivuljom n(t) = a * e^(b * t) po regijama. Krivulja aproksimira podatke zadnjih 7 dana, izračunate iz podataka objavljenih na koronavirus.hr, koristeći broj stanovnika po županijama iz 2019. s dzs.hr. Krivulja se prikazuje ukoliko je R^2 aproksimacije > 0.95, prikazana je narančastom bojom. Zelena krivulja prikazuje vrijednosti aproksimacijske krivulje 7 dana u budućnosti (deblja linija), dok su tanjom zelenom linijom prikazane vrijednosti krivulje do dana u kojem bi vrijednost mogla doći do praga od 75/200 zaraženih na 100k stanovnika. Generirano automatski, nakon objave službenih podataka na koronavirus.hr.)
